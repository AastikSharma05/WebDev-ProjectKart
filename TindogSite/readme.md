# Tindog

#### Techstack

This website is made using very basic techstack and those are:

* HTML5
* CSS3
* Boostrap4

#### Description

This site was built taking motivation from Tinder which is a dating app for human beings but this is for dogs. Dogs are the most loved creatures on earth,people treat them like their own children and every parent want their children to settle at some point of time.
This webapp will come into play at that time. This will give every dog (male/female) the liberty to choose their "love of their dreams"
